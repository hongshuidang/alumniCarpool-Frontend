# alumniCarpool

## Clone source code
```
Git clone https://github.com/hongshuidang/alumniCarpool-Frontend.git
```

## Project setup: install dependencies
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
